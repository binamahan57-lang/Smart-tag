# Smart Tag Title Sync

این پروژه یک اپ اندروید ساده است که:
- یک پوشه انتخاب می‌کند
- همه فایل‌های MP3 داخل آن را پیدا می‌کند
- نام فایل را به عنوان `Title` داخل تگ ID3 می‌نویسد
- در صورت فعال بودن گزینه، شماره/پیشوند اول نام فایل را هم حذف می‌کند

## نکته
در این محیط APK واقعی ساخته نشده، اما سورس کامل آماده است و در Android Studio قابل باز شدن است.

## ساخت APK
1. پروژه را در Android Studio باز کن
2. صبر کن Gradle Sync انجام شود
3. از منوی `Build` گزینه `Build Bundle(s) / APK(s)` را بزن
4. `Build APK(s)` را انتخاب کن

اگر خواستی، در پیام بعدی می‌توانم نسخه‌ای هم بدهم که:
- به‌جای انتخاب پوشه، فقط چند فایل را بگیرد
- یا پیشوندهای بیشتری را حذف کند
- یا خروجی CSV گزارش بدهد
